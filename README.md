## Data Format

### 4-Panel

| Field    | Type     | Value                    |
| -------- | -------- | ------------------------ |
| id       | string   | ID (index)               |
| title    | string   | Title                    |
| students | string[] | Student name (ja)        |
| href     | string   | Tweet or Official URL    |
| deleted? | boolean  | Flag for deleted 4-panel |

#### Example

```json
[
  {
    "id": "41",
    "title": "折衷案",
    "students": ["エイミ", "ヒマリ"],
    "href": "https://bluearchive.jp/comics/41/1",
    "deleted": true
  },
  {
    "id": "42",
    "title": "勇気のポーション",
    "students": ["シグレ", "ノドカ"],
    "href": "https://twitter.com/blue_archivejp/status/1465531285523542017"
  }
]
```

### Student

| Field  | Type           | Value     |
| ------ | -------------- | --------- |
| id     | string         | ID (name) |
| ja     | string         | JA name   |
| en     | string         | EN name   |
| school | string \| null | School ID |

#### Example

en.json => EN key

```json
{
  "hanako": {
    "id": "hanako",
    "ja": "ハナコ",
    "en": "Hanako",
    "school": "trinity"
  }
}
```

ja.json => JA key

```json
{
  "コハル": {
    "id": "koharu",
    "ja": "コハル",
    "en": "Koharu",
    "school": "trinity"
  }
}
```
